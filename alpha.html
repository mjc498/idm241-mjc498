RULES

Like - Default

  • Hover ON
    • Button opacity darkens 25%
    • 2 second delay "I like this" appears as tagline
  • Hover OFF
    • Button opacity changes to default gray color
    • "I like this" disappears immediately
  • Mouse Down
    • Button opacity darkens to 50%
    • "I like this" disappears immediately
  • Mouse Up
    • Icon outline blinks with various colors and effects
    • Bouncy animation plays
    • Button opacity changes to 25%
    • Database record flag set to TRUE

Like - Active

  • Hover ON
    • Button opacity darkens 25%
    • 2 second delay "Unlike" appears as tagline
  • Hover OFF
    • Button opacity changes to default gray color
    • "Unlike" disappears immediately
  • Mouse Down
    • Button opacity darkens to 50%
    • "Unlike" disappears immediately
  • Mouse Up
    • Icon outline fills in with default gray color
    • Database record flag set to FALSE
