RULES

Container

  • Hover ON
    • Shadow appears at bottom edge of container
  • Hover OFF
    • Shadow disappears at bottom edge of container

Connect - Default

  • Hover ON
    • Cursor changes to finger
    • Button opacity changes to 15% blue via slide-in animation
  • Hover OFF
    • Cursor changes back to pointer
    • Button opacity changes to default white color
  • Mouse Down
    • Button changes to 100% light blue
  • Mouse Up
    • Button text changes to "Pending..."
    • Database record flag set to TRUE

Connect - Active

  • Hover ON
    • Cursor changes to finger
    • Button opacity changes to 15% blue via slide-out animation
  • Hover OFF
    • Cursor changes back to pointer
    • Button opacity changes to default white color
  • Mouse Down
    • Button changes to 100% light blue
  • Mouse Up
    • Button text changes back to "Connect +"
    • Database record flag set to FALSE

"X" Button

  • Hover ON
    • Cursor changes to finger
    • "X" rotates forward 180 degrees
  • Hover OFF
    • Cursor changes back to pointer
    • "X" rotates backward 180 degrees
  • Mouse Down
    • Button opacity changes to white
  • Mouse Up
    • Button opacity changes back to default transparency
    • Button slides to the right then bounces to the left
    • Container fades out while the new container slides in

View Profile

  • Hover ON
    • Cursor changes to finger
    • Drop shadow appears at bottom of profile picture
    • "View Profile" appears at top of profile picture
  • Hover OFF
    • Cursor changes back to pointer
    • Drop shadow disappears from bottom of profile picture
    • "View Profile" disappears from top of profile picture
  • Mouse Down
    • Inner shadow appears at top of profile picture
  • Mouse Up
    • Page blurs as loading animation plays before user is taken to a new tab
